# labwep 🧪

[![Vercel Deployment](https://img.shields.io/badge/Deploy-Vercel-black?style=for-the-badge&logo=vercel)](https://labwep.vercel.app)
[![React](https://img.shields.io/badge/React-19-blue?style=for-the-badge&logo=react)](https://react.dev/)
[![Three.js](https://img.shields.io/badge/Three.js-Background-orange?style=for-the-badge&logo=three.js)](https://threejs.org/)

**labwep** is a premium, professional lab report generator designed to help students and researchers create flawlessly formatted PDFs in seconds. Built with a focus on aesthetics and ease of use, it bridges the gap between raw data and academic excellence.

🔗 **Live Demo**: [https://labwep.vercel.app](https://labwep.vercel.app)

---

## ✨ Key Features

- **🚀 Instant PDF Generation**: Export professional-grade lab reports matching GUB LaTeX templates with a single click.
- **🐍 Multi-Language Support**: Dedicated support for **Python** and **R Language** code blocks with IDE-quality syntax highlighting.
- **🎨 Advanced Theming**: 12+ professional code themes including *Dracula*, *Nord*, *Night Owl*, and *R Studio Classic*.
- **📊 Dynamic Content**: easily add tables, image outputs, objectives, and conclusions with intuitive form controls.
- **🌀 Modern Aesthetics**:
  - Dynamic **Three.js** particle background.
  - Vibrant **Mesh Gradient** design system.
  - Premium **Glassmorphism** (frosted-glass UI).
- **📱 Responsive Design**: Fully optimized for mobile formatting and generation.
- **🤖 Smart Filenames**: Automatic professional naming pattern: `id__coursecode_lab{reportno}.pdf`.

---

## 🛠️ Technology Stack

- **Frontend**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Visuals**: [Three.js](https://threejs.org/) (3D Particles)
- **Icons**: [Lucide React](https://lucide.dev/)
- **PDF Engine**: [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/)
- **Syntax Highlighting**: [Prism.js](https://prismjs.com/) via `react-syntax-highlighter`
- **Animations**: [AOS](https://michalsnik.github.io/aos/) (Animate on Scroll)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tanvir0106/labwep.git
   cd labwep
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
